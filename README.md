# challenge_literatura
Challenge de programación Java desarrollado en el marco del programa **ONE (Oracle Next Education)** de **Alura Latam**.

---

## 🗂️ Descripción

Es una aplicación de consola que permite construir un catálogo de libros interactivo. La aplicación consume la API pública [Gutendex](https://gutendex.com) para buscar libros, almacena los resultados en una base de datos PostgreSQL y ofrece múltiples opciones de consulta al usuario.

---

## ✨ Funcionalidades

| Opción | Descripción |
|--------|-------------|
| `1` | Buscar libro por título (consume la API Gutendex) |
| `2` | Listar todos los libros registrados (ordenados por descargas) |
| `3` | Listar todos los autores registrados (orden alfabético) |
| `4` | Listar autores vivos en un determinado año |
| `5` | Listar libros por idioma |
| `6` | Ver estadísticas del catálogo |
| `0` | Salir de la aplicación |
